# FellowShift

Blind typing (touch typing) language learning app: type using your **OS keyboard layout** (switch layouts for RU/HU as needed). FellowShift reads your physical key codes and the character you typed.

## Tech
- Next.js 16 (App Router) + TypeScript
- Tailwind CSS
- Zustand (game/session state)
- Framer Motion (feedback animations)

## Run
1. Install deps: `npm install`
2. Start dev: `npm run dev`
3. Open: `http://localhost:3000`

## Play route
- `http://localhost:3000/play/lotr-ru/bag-end-1`
